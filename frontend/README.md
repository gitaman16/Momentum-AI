# Frontend (React + TypeScript + Tailwind)

## Run locally

```bash
cp .env.example .env   # set VITE_API_URL and VITE_GOOGLE_CLIENT_ID
npm install
npm run dev
```

Opens on `http://localhost:5173`.

## Pages

- **Dashboard** - stats, AI daily plan, upcoming deadlines
- **Tasks** - kanban board with AI auto-schedule
- **Goals** - create goals, AI decomposition into tasks
- **Analytics** - completion rate, status/priority charts, estimation accuracy
- **Assistant** - risk analysis, weekly review, insight history
